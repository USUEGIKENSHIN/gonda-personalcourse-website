ELEMENTS OF WINDOWING SYSTEM

DEVICE INDEPENDENCE

A device independence is the process of making a software application able to function on a wide variety of devices regardless of the operating system and local hardware on which the software is used.
EXAMPLES: Programming Language, applications, API

SHARED RESOURCE

In computing, a shared resource, or network share, is a computer resource made available from one host to other hosts on a computer network. It is a device or piece of information on a computer that can be remotely accessed from another computer.
EXAMPLES: Printer, network, display, memory, storage, and application
